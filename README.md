# 工厂车间管理系统 java28

## 项目概述

工厂车间管理系统是一个基于B/S架构的MVC模式系统，采用前后端分离的设计，前端使用Vue.js框架，后端采用Java语言及SpringBoot框架，旨在为工厂车间提供全面的管理解决方案。

## 技术栈

- **系统架构**: B/S、MVC
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术框架**: 
  - Java
  - MySQL
  - SpringBoot
  - Mybatis
  - Vue
